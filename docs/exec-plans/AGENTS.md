# exec-plans/ — Agent 地图

> 本目录存放**执行计划**和**关键 Bug 根因记录**。

---

## 文件清单

| 文件 | 内容 | 什么时候需要看 |
|------|------|-------------|
| `critical-bugs-resolved.md` | 已解决的关键 Bug 根因表（按时间倒序） | **遇到奇怪问题先查这个**，避免重复踩坑 |
| `tech-debt-tracker.md` | 技术债务追踪表 | 计划大规模重构前看 |

---

## 规则

- 新 Bug 修复后，如果根因隐蔽或可能复现，追加到 `critical-bugs-resolved.md`
- 不要为每个 Bug 建单独文件，全部累积在同一个文档里
